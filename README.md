
# Student Productivity & Performance Analysis (Excel)

## Objective
Analyze how study habits and lifestyle factors such as sleep hours and extracurricular
activities impact student performance.

## Tools Used
- Microsoft Excel
- Excel formulas (IF, AVERAGEIF, COUNTIF)
- Pivot Tables
- Charts & Slicers

## Dataset
Student productivity dataset containing study hours, sleep hours, previous scores,
extracurricular activities, and performance index.

## Analysis Performed
- Data cleaning and preparation
- Feature creation (performance and sleep categories)
- Formula-based analysis
- Pivot table summaries
- Interactive dashboard with slicers
